Textures from SkyboxSet: https://93i.de/en/downloads

These textures are downsampled to 512x512 pixels, but are
otherwise unmodified w.r.t. the originals (i.e., no flipping).

The files have been renamed to follow the CMFT naming convention:
* CloudyLightRaysLeft.png -> CloudyLightRays_posx.png
* CloudyLightRaysRight.png  -> CloudyLightRays_negx.png
* CloudyLightRaysUp.png -> CloudyLightRays_posy.png
* CloudyLightRaysDown.png -> CloudyLightRays_negy.png
* CloudyLightRaysFront.png -> CloudyLightRays_posz.png
* CloudyLightRaysBack.png -> CloudyLightRays_negz.png
